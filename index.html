Loja de Uniformes - Dashboard Admin

Esse é o projeto de um dashboard administrativo para uma Loja de Uniformes. Ela foi criada para atender a demanda da Silvana Uniformes.

Seu FrontEnd foi desenvolvido em Nuxt 3, um Vue com super poderes. Utilizei a biblioteca de componentes PrimeVue, Tailwind para estilos e TypeScript para tipagem. Além disso, utilizei o Vuelidade para validação de formulários e Pinia para gerenciameto de estados.

Seu Backend foi desenvolvido em Java com Spring Framework para criação da API, JPA para persistência de dados e PostgreSQL como banco de dados.

A API possui sistema de autenticação e autorização ( Spring Security ), testes unitários com JUnit5 e documentação com Swagger.

O sistema permite ao usuário cadastrar empresas, produtos e usuários. Além disso, é possível visualizar, editar e excluir esses dados. Por fim, também tem um sistema de relatórios para acompanhar e medir todos os dados relevantes.

Auto Cars - Aluguel e Venda de Carros

O projeto Auto Club é uma aplicação de aluguel e venda de carros, onde o usuário pode visualizar os carros disponíveis, alugar e comprar.

Ele foi desenvolvido com ReactJS e Google Firebase. Os dados estão armazenados na Firestore e são buscados utilizando JavaScript puro. O sistema de autenticação e autorização é feito pelo módulo de Authentication do Firebase.

O principal da plataforma são suas funcionalidades de CRUD para anúncios de carros, aluguel e compra de veículos.

Os estados são gerenciados com ContextAPI.

Behold Travel - Blog de Viagens

Behold Travel é um blog onde qualquer pessoa pode criar artigos descrevendo sobre suas aventuras! O projeto foi desenvolvido em ReactJS e Google Firebase.

Ele possui sistema de autenticação e autorização, o usuário logado pode criar, editar e excluir seus artigos.

Além disso, o usuário pode deixar comentários em outros artigos e visualizar os artigos de outros usuários. Os artigos mais recentes são exibidos na página inicial.
